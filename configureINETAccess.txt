1) Configure DHCP client on the default interface (e.g. ens33)
  # dhclient ens33

2) Edit the main network config file to make sure networking is enabled
  # vim /etc/sysconfig/network
    ... NETWORKING=yes

3) Edit network config file for (ens33) to enable it to use a dynamic IP using DHCP
  # vim /etc/sysconfig/network-scripts/ifcfg-ens33
  ...DEVICE=ens33
  ...BOOTPROTO=dhcp
  ...ONBOOT=yes
  ...NM_CONTROLLED=no

4) Restart network service
  #service network restart

5) Check that ens33 has pulled an IP address
  #ifconfig
