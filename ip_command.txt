##ifconfig, arp, netstat, and route commands have been deprecated

##The 'ip' command replaces all of these deprecated commands.

|Obsolete Command| New RHEL 7 Command       | Details                              |
|  -----------   |  ----------------------- | -------------------------------------
| ifconfig       | ip [-s] link, ip addr    | Displays the link status and address info for all network interfaces|
| ifconfig eth0 <IP> -> netmask <netmask> | ip addr add <IP/24> dev eth0 | Assigns an IP addr & netmask to the eth0 interface|
| arp | ip neigh | Display ARP table (neigh short for neighbor)|
|route, netstat -r | ip route | Display routing table |
|netstat -tupanl | ss -tupan | List all listening & non-listening sockets along with associated PIDs|
